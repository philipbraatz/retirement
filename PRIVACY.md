# Privacy

This is a personal side project. The code does not intentionally collect, store, or transmit personal data.

If you choose to use real financial data:
- Keep it local and out of version control.
- Do not share sensitive data in issues or pull requests.
- Consider using sample or anonymized values.

Telemetry
- No telemetry is collected by this project.
- Third-party dependencies may have their own policies (e.g., ScottPlot). Review their documentation before enabling optional features.
