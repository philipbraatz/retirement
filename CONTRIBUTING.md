# Contributing

Thanks for your interest in improving Retirement Planner!

How to contribute
- Open an issue describing the change or idea.
- Fork the repo and create a feature branch.
- Keep PRs small and focused. Include tests when reasonable.

Development
- Requires .NET 8 SDK.
- Build: `dotnet build`
- Test: `dotnet test`
- Run console app: `dotnet run --project .\RetirementPlanner.Console`

Coding
- Enable nullable reference types and treat warnings as errors when possible.
- Add XML docs to public APIs when helpful.

Legal
- By submitting a pull request you agree your contribution will be licensed under the repository’s `LICENSE` (MIT).

Code of Conduct
- Participation is governed by `CODE_OF_CONDUCT.md`.
