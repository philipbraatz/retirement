# Code of Conduct

This project follows the Contributor Covenant Code of Conduct (v2.1).

- Our Pledge: Foster an open and welcoming environment.
- Our Standards: Be respectful, constructive, and inclusive.
- Enforcement: Maintainers will enforce this code; unacceptable behavior may result in temporary or permanent restrictions.

For the complete text, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

To report issues, please contact the maintainers privately or open a minimal public issue requesting a private contact channel.
